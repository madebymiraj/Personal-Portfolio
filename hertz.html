Web & Mobile Style Guide

I developed an internal web and mobile style guide for Hertz to use as a resourceful collaboration tool to help bridge the design and development teams by allowing for a more efficient workflow between the design an development processes.

This global web and mobile style guide now serves as a foundation of various UI components that are broken down with guidelines and accompanied by their markup code as suggested. This allows for reusability of code for front-end development, and standardizes how each component should be used in order to maintain consistency with Hertz's brand amongst all the Hertz divisions. The documented components also allow for faster adaptability for future design and technological changes.

Car Rental Reservation Redesign for Responsive Web

I worked with the design, development, and business analyst teams to redesign the responsive web car rental reservation process to redefine how our users navigate and reserve car rentals on their mobile devices. We used the, now approved, web and mobile style guide and the user testing conducted by the quality assurance teams to improve the current reservation process and make it consistent with the Hertz brand by making modifications to the structure and the user interface components.

Android Mobile App Redesign

Worked with the design team to redesign the Hertz Android mobile app. We conducted a claims analysis between several other Android mobile apps to help introduce better solutions and improvements to improve the app's usability. While doing so, we stayed consistent with the style guide to introduce components that our users are already familiar with while using Hertz's services.
